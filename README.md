# 🔐 Caesar Cipher Encryption & Decryption (Assembly x86)

โปรเจคนี้คือโปรแกรมจำลองการเข้ารหัส (**Encryption**) และการถอดรหัส (**Decryption**) ข้อมูลตัวอักษรโดยใช้วิธี **Caesar Cipher** (การเลื่อนตัวอักษร) พัฒนาด้วยภาษา Assembly สำหรับสถาปัตยกรรม 8086 โดยรันบน DOS environment

## 📋 รายละเอียดโปรเจค
โปรแกรมจะรับข้อความจากผู้ใช้ (Input String) จากนั้นจะทำการเลื่อนตำแหน่งตัวอักษรไปข้างหน้า 3 ตำแหน่ง (Shift = 3) เพื่อทำการเข้ารหัส และทำการถอดรหัสกลับมาเป็นข้อความเดิมเพื่อแสดงผลการทำงานที่ถูกต้อง

### ฟีเจอร์หลัก:
- **Input Handling:** รับข้อความจาก Keyboard ได้สูงสุด 100 ตัวอักษร
- **Caesar Cipher Logic:** รองรับการเลื่อนตัวอักษรทั้งตัวพิมพ์เล็ก (a-z) และตัวพิมพ์ใหญ่ (A-Z)
- **Wrap-around:** มีระบบวนลูปตัวอักษร (เช่น ถ้า 'Z' + 3 จะวนกลับมาเป็น 'C')
- **Data Integrity:** แสดงผลทั้งข้อความที่เข้ารหัสแล้ว และข้อความที่ถอดรหัสกลับมา เพื่อยืนยันความถูกต้อง

---

## 👥 โครงสร้างสมาชิกและหน้าที่รับผิดชอบ

โปรเจคนี้แบ่งการทำงานออกเป็น 3 ส่วนหลักตามโครงสร้าง Code:

### 1. ALONGKOT (Main Execution Flow)
* ควบคุมลำดับการทำงานหลัก (Main Loop)
* จัดการการวนลูปเพื่ออ่านตัวอักษรทีละตัวจาก Buffer
* ควบคุมการเรียกใช้ฟังก์ชัน Logic และการแสดงผลลัพธ์ออกทางหน้าจอ

### 2. JIRAPHAT (Mathematical Logic)
* ออกแบบฟังก์ชัน `encrypt_logic` และ `decrypt_logic`
* คำนวณการ Shift ค่า ASCII
* จัดการเงื่อนไขขอบเขต (Boundary Checking) เพื่อไม่ให้กระทบกับสัญลักษณ์พิเศษและจัดการการวนรอบตัวอักษร (Wrap around 26 ตัว)

### 3. NOPPARAT (Memory & String Management)
* จัดการส่วนประกาศตัวแปร (Data Segment)
* กำหนดขนาด Buffer สำหรับรับข้อมูลและเก็บข้อมูลหลังประมวลผล
* จัดการการแสดงผลข้อความแจ้งเตือน (Prompt messages)

## 🛠 ข้อมูลทางเทคนิค (Technical Details)

* **Platform:** DOS (8086 Assembly)
* **Format:** `.COM` file (using `org 100h`)
* **Interrupts used:**
    * `int 21h, ah 09h`: ใช้แสดงผล String
    * `int 21h, ah 0Ah`: ใช้รับ Input แบบ Buffered
    * `int 21h, ah 4Ch`: ใช้คืนทรัพยากรและจบการทำงาน
* **Encryption Key:** Shift = 3

## 🚀 วิธีการใช้งาน

1.  คัดลอก Code ไปวางในโปรแกรม **EMU8086** หรือโปรแกรมจำลอง DOS อื่นๆ
2.  ทำการ **Compile** และ **Run**
3.  เมื่อโปรแกรมขึ้นข้อความ `Input original text: ` ให้พิมพ์ข้อความที่ต้องการ
4.  โปรแกรมจะแสดงผลลัพธ์ 2 บรรทัด:
    * **Phase 1:** ข้อความที่ถูกเข้ารหัสแล้ว
    * **Phase 2:** ข้อความที่ถูกถอดรหัสกลับมา (ต้องตรงกับที่เราพิมพ์ไป)
